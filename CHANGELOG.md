# Changelog

All notable changes to SecureGate will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-08

### Added
- Initial release of SecureGate
- Vulnerability scanning using OSV Scanner
- License detection and compliance checking
- Configurable policy enforcement (block/annotate modes)
- Severity-based filtering (INFO, LOW, MEDIUM, HIGH, CRITICAL)
- Automatic PR commenting
- JSON and Markdown report generation
- Artifact upload support
- Whitelist and ignore list functionality
- Comprehensive documentation and examples

### Features
- Support for Flutter and Dart packages
- Integration with GitHub Actions
- Customizable security policies via YAML configuration
- Detailed vulnerability reporting with references
- License violation detection
